## 시작 가이드

### Requirements

- [ojdbc6.jar](https://mvnrepository.com/artifact/oracle/ojdbc6/11.2.0.3)
- [JavaSE-17(zulu-17)](https://www.azul.com/downloads/#zulu)
### Installation
``` bash
$ git clone https://github.com/Ohsaam/ChatJava.git
$ cd ChatJava
```
